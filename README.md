# AI Powered Financial Advisor

An AI-powered financial assistant that helps users manage expenses, view financial insights, and plan smarter budgeting strategies.

## Live Demo

Frontend: https://ai-powered-financial-advisor-two.vercel.app/
Backend API: https://ai-powered-financial-advisor.onrender.com

## Features

• Expense tracking interface
• Budget planning dashboard
• Financial insights and saving tips
• Interactive charts and analytics
• AI chatbot assistant (currently placeholder response)

## Tech Stack

Frontend: React + Tailwind CSS
Backend: Flask API
Deployment: Vercel (Frontend) + Render (Backend)
